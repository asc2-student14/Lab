# Troubleshooting Guide

## Common Issues

### Coffee Tastes Bitter
**Cause**: Over-extraction  
**Solutions**:
- Use coarser grind setting
- Reduce extraction time
- Lower water temperature by 2-3°C
- Check bean freshness (optimal: 7-21 days post-roast)

### Coffee Tastes Sour  
**Cause**: Under-extraction  
**Solutions**:
- Use finer grind setting
- Increase extraction time
- Raise water temperature by 2-3°C
- Ensure proper dose weight (18g ±0.5g)

### Weak Crema
**Causes & Solutions**:
- **Stale beans**: Use beans 7-21 days post-roast
- **Grind too coarse**: Adjust to finer setting
- **Low pressure**: Check system pressure (target: 9 bar)
- **Incorrect dose**: Ensure 18g dose weight

### Milk Won't Steam Properly
**Solutions**:
- Clean steam wand thoroughly
- Check milk freshness (use within 3 days of opening)
- Use cold milk (4°C) for best results
- Ensure milk fat content ≥2%

### Robot Arm Movement Issues
**Safety First**: Stop operation immediately  
**Checks**:
- Verify no obstructions in work area
- Run calibration sequence
- Check emergency stop is disengaged
- Inspect cables for damage

## Error Codes

| Code | Description | Action |
|------|-------------|---------|
| E001 | Insufficient beans | Refill bean hopper |
| E002 | Water reservoir empty | Refill water reservoir |
| E003 | Grinder malfunction | Clear jam, restart system |
| E004 | Temperature too low | Wait for heating, check sensors |
| E005 | Milk reservoir empty | Refill milk reservoir |
| E006 | Steam wand blocked | Clean steam wand |
| E007 | Scale calibration error | Run calibration sequence |

## Maintenance Schedule

### Daily
- Empty drip tray
- Wipe exterior surfaces  
- Check consumable levels

### Weekly
- Deep clean steam wand
- Calibrate scale
- Inspect robotic arm joints

### Monthly  
- Descale boiler system
- Replace water filter
- Update recipe parameters
